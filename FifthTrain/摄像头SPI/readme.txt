实验器材:
	探索者STM32F4开发板
	
	
硬件资源:
	1,DS0(连接在PF9)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,ALIENTEK 2.8/3.5/4.3/7寸TFTLCD模块(通过FSMC驱动,FSMC_NE4接LCD片选/A6接RS)
	4,按键KEY0(PE4)/KEY1(PE3) 
	5,DCMI接口(用于驱动OV2640摄像头模块)
	6,定时器3(用于打印摄像头帧率等信息)

	7,ALIENTEK OV2640摄像头模块,连接关系为:  
		OV2640模块 ------------ STM32开发板


		 OV_D0~D7  ------------  PE6/PE5/PB6/PC11/PC9/PC8/PC7/PC6
		 OV_SCL    ------------  PD6
		 OV_SDA    ------------  PD7
		 OV_VSYNC  ------------  PB7
		 OV_HREF   ------------  PA4
		 OV_RESET  ------------  PG15
		 OV_PCLK   ------------  PA6
		 OV_PWDN   ------------  PG9 

USART1 A9 A10
USART2 A2 A3

LCD
D0,1,4,5,8,9,10,14,15
B15
PE7~15
PF12
G12

M8266WIFI模块默认的连接关系为
		 SPI1_SCK  ------------  PA5
		 SPI1_MISO   ------------  PA6
		 SPI1_MOSI   ------------  PA7 
		 SPI1_nCS    ------------  PA4
		 SPI1_nRESET ------------  PB14 

修改后的连接关系为（为方便保持SPI1不变）
		 SPI1_SCK  ------------  PA5
		 SPI1_MISO   ------------  PB4(没问题)
		 SPI1_MOSI   ------------  PA7 
		 SPI1_nCS    ------------  PE1(没问题)
		 SPI1_nRESET ------------  PB14 

QVGA格式，图像大小320*240下，发送一帧图像是0.3s，采集一帧数据的时间是1.4s。
QCIF格式，图像大小176*144下，发送一帧图像是0.2s，采集一帧数据的时间是2.05s。
QQVGA格式，图像大小160*120下，发送一帧图像是0.2s，采集一帧数据的时间是2.3s。
CIF格式，图像大小352*288下，发送一帧图像是0.3s，采集一帧数据的时间是1.5s。
